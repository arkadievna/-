git init и git clone легко спутать, поскольку обе команды можно использовать для инициализации нового репозитория Git. Однако команда git clone зависит от git init ипредназначена для создания копии существующего репозитория. Выполнение git clone сначала вызывает git init, чтобы создать новый репозиторий, затем копирует данные из существующего репозитория и извлекает новый набор рабочих файлов.
