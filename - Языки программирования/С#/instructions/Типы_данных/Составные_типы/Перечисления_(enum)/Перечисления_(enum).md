ПЕРЕЧИСЛЕНИЯ (enum) - это НЕ коллекция
перечисления делают код более читаемым, более удобным
```csharp
enum En
{
    one,
    two
}
```

хотим сделать метод, который будет выводить название дня взависимости от его номера
```csharp
// определение перечисления "enum Days" должно быть вынесено за пределы метода Main
enum Days
{
    // каждый элемент перечисления должен быть разделен запятыми
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday,
    Sunday
}

static void Main(string[] args)
{
    // вызов метода WriteLine должен находиться внутри метода Main
    Console.WriteLine(Days.Sunday);
    Console.ReadLine();
}
```
усовершенствованный код
```csharp
enum Days
{
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday,
    Sunday
}

static void Main(string[] args)
{
    Console.WriteLine("Введите номер дня недели (1-7): ");
    int input = Convert.ToInt32(Console.ReadLine()?? "0");

    // Преобразуем введенное значение в тип перечисления Days
    Days day = (Days)input; // `(Days)input` является примером явного приведения типа
    // в данном случае, оно используется для преобразования введенного значения `input` типа `int` 
    // в указанный тип перечисления `Days`
// Типы переменной `input` и перечисления `Days` различаются, поэтому необходимо выполнить приведение типа
// В данном случае, мы приводим значение `input` к типу `Days` с помощью оператора приведения типа `(Days)`, 
// чтобы его можно было использовать в дальнейшем в программе как элемент перечисления
// Затем, приведенное значение сохраняется в переменной `day` типа `Days`, чтобы использовать его при выводе названия дня недели.

    // Выводим название дня недели
    Console.WriteLine(day);
    Console.ReadLine();
}
```
